IP Displayer API

An easy way to get a clients IP address! Deploy and serve!

  • CORS (Cross Origin) Allowed - Implement JS Code to fetch Users IP using our API
  • Cache Disabled so you never fetch old a user's old IP address
  • Popular Formats Supported - Open to adding more


Formats Allowed:
  • JSON - "/api?format=json"
  • XML - "/api?format=lxml"
  • HTML - "/api?format=html"
